# 04-费用管理

## 功能名

费用管理

## 需求描述

统一管理门诊和住院产生的所有账单，支持病人查看账单明细、模拟缴费，管理员查看全部账单和支付记录。费用模块是门诊和住院的终点。

## 概述

门诊处方提交时自动生成账单，住院出院时生成最终账单。病人可在费用管理页查看待缴账单并完成模拟支付，支付后账单和处方状态同步更新。管理员可查看全部账单和支付流水。

## 相关页面设计

### 1. 我的账单页（病人端）

**上方指标卡片**：

| 卡片 | 说明 |
|------|------|
| 待缴金额（元） | 红色，所有 UNPAID 账单合计 |
| 已缴金额（元） | 绿色，所有 PAID 账单合计 |
| 账单总数 | |

**表格列**：账单号、来源标签、金额（元）、状态标签、创建时间

**来源标签**：`OUTPATIENT`(蓝)、`INPATIENT`(紫)

**状态标签**：`UNPAID`(红)、`PAID`(绿)

**筛选**：状态、来源类型、日期范围

**点击行展开**：费用明细列表（类型标签、名称、单价、数量、小计）

**类型标签**：`CONSULTATION`(蓝)、`MEDICINE`(绿)、`BED`(橙)、`TREATMENT`(紫)

**缴费按钮**：UNPAID 账单旁显示"去缴费"。

**缴费确认弹窗**：展示账单明细汇总 → "确认支付" → 模拟支付完成。

### 2. 全部账单页（管理员端）

**表格列**：账单号、病人、来源标签、金额（元）、状态标签、支付时间、创建时间

**筛选**：病人姓名、状态、来源类型、日期范围

**展开行**：费用明细

**操作**：仅查看，不缴费。

### 3. 支付记录页（管理员端）

**表格列**：支付流水号、账单号、病人、金额（元）、支付方式、支付时间

**支付方式标签**：`ONLINE`(蓝)、`CASH`(灰)

**筛选**：病人姓名、日期范围

### 4. 我的支付记录页（病人端）

**表格列**：支付流水号、账单号、金额（元）、支付方式、支付时间

**筛选**：日期范围

## 用户旅程

### 门诊缴费
医生开具处方 → 系统自动生成账单 B001(UNPAID, 86元) → 病人登录 → 费用管理 → 待缴金额显示 86 元 → 点击账单 B001 → 展开明细：诊疗费 30 + 阿莫西林 56 → 点击"去缴费" → 确认弹窗展示合计 86 元 → 点击"确认支付" → Toast "支付成功" → 账单变 PAID → 处方变 PAID → 生成支付记录。

### 住院出院补缴
出院结算时 prepaid_balance = -200 → 生成补缴账单 B002(UNPAID, 200元) → 病人缴费 → 账单 PAID → 住院档案 DISCHARGED。

## 用户故事

1. 作为病人，我希望一眼看到还有多少待缴费用，避免遗漏。
2. 作为病人，我希望缴费前能看到每笔账单的明细，确认无误再支付。
3. 作为管理员，我希望查看全部账单和支付流水，掌握整体收入。
4. 作为病人，我支付后希望处方状态同步更新。

## 实现逻辑

### 账单生成

**门诊账单**（处方创建时自动生成）：
```
INSERT bills (bill_no, patient_id, source_type='OUTPATIENT',
  source_id=prescription.id, total_amount, status='UNPAID')

INSERT bill_items:
  - CONSULTATION: item_name='诊疗费', unit_price=consultation_fee,
    quantity=1, amount=consultation_fee
  - MEDICINE: 各药品 item_name, unit_price, quantity, amount
```

**住院账单**（出院结算时生成）：
```
INSERT bills (bill_no, patient_id, source_type='INPATIENT',
  source_id=admission.id, total_amount, status='PAID')

INSERT bill_items:
  - BED: ward.daily_charge × 住院天数
  - MEDICINE: 各住院记录药品汇总
  - TREATMENT: 各住院记录治疗项目 + 每日 treatment_fee
```

### 支付流程（同一事务）
```
a. 校验 bill.status = 'UNPAID'
b. INSERT payments (payment_no, bill_id, patient_id, amount,
   payment_method='ONLINE', paid_at=NOW())
c. UPDATE bills SET status='PAID', paid_at=NOW()
d. IF bill.source_type='OUTPATIENT':
     UPDATE prescriptions SET status='PAID'
     WHERE id = bill.source_id
```

## 功能细节描述

- 支付流水号格式 `PAY{yyyyMMdd}{4位序号}`，系统自动生成
- 同一账单不可重复支付（校验 status）
- 支付记录不可删除/修改
- 账单金额为 0 时自动标记 PAID，不生成支付记录
- 病人只能查看和支付自己的账单（数据权限）
- 支付方式当前固定 ONLINE
- 支付确认弹窗含"取消"按钮
