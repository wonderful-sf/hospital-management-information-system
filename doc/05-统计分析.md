# 05-统计分析

## 功能名

统计分析

## 需求描述

提供科室排班、医生工作量、病人治疗情况三类统计查询，满足课程实验要求，以表格和 CSS/JS 条形图展示，不引入图表库。

## 概述

统计页面面向管理员和医生（医生仅查看自己的数据）。将排班、工作量、治疗三类核心指标以可视化方式呈现，适用于课程答辩演示和日常管理。首页工作台嵌入核心指标卡片。

## 实现逻辑

### 科室排班统计
```sql
SELECT d.name AS department_name,
  SUM(CASE WHEN ds.schedule_type = 'OUTPATIENT' THEN 1 ELSE 0 END)
    AS outpatient_count,
  SUM(CASE WHEN ds.schedule_type = 'INPATIENT_ROUND' THEN 1 ELSE 0 END)
    AS inpatient_round_count,
  COUNT(*) AS total_count
FROM departments d
LEFT JOIN doctor_schedules ds ON d.id = ds.department_id
WHERE ds.start_time BETWEEN #{startDate} AND #{endDate}
GROUP BY d.id
ORDER BY total_count DESC
```

### 医生工作量统计
```sql
SELECT doc.name AS doctor_name, d.name AS department_name,
  (SELECT COUNT(*) FROM outpatient_visits ov
   WHERE ov.doctor_id = doc.id
   AND ov.visited_at BETWEEN #{startDate} AND #{endDate})
   AS visit_count,
  (SELECT COUNT(*) FROM admissions adm
   WHERE adm.attending_doctor_id = doc.id
   AND adm.status = 'ACTIVE') AS admission_count,
  (SELECT COUNT(*) FROM prescriptions p
   WHERE p.doctor_id = doc.id
   AND p.created_at BETWEEN #{startDate} AND #{endDate})
   AS prescription_count,
  (SELECT COALESCE(SUM(p.total_amount), 0) FROM prescriptions p
   WHERE p.doctor_id = doc.id
   AND p.created_at BETWEEN #{startDate} AND #{endDate})
   AS total_prescription_amount
FROM doctors doc
JOIN departments d ON doc.department_id = d.id
WHERE doc.status = 'ACTIVE'
ORDER BY visit_count DESC
```

医生端额外查询今日信息：
```sql
-- 今日排班
SELECT start_time, end_time FROM doctor_schedules
WHERE doctor_id = #{doctorId} AND DATE(start_time) = CURDATE()

-- 待接诊数
SELECT COUNT(*) FROM registrations
WHERE doctor_id = #{doctorId} AND status = 'REGISTERED'
AND DATE(registered_at) = CURDATE()
```

### 病人治疗情况统计
```sql
SELECT p.name, p.medical_record_no,
  COUNT(DISTINCT r.id) AS outpatient_count,
  COUNT(DISTINCT a.id) AS admission_count,
  COALESCE(SUM(b.total_amount), 0) AS total_cost
FROM patients p
LEFT JOIN registrations r ON p.id = r.patient_id
  AND r.registered_at BETWEEN #{startDate} AND #{endDate}
LEFT JOIN admissions a ON p.id = a.patient_id
LEFT JOIN bills b ON p.id = b.patient_id AND b.status = 'PAID'
GROUP BY p.id
ORDER BY total_cost DESC
```

### 条形图实现
```css
.bar-container { background: #f0f0f0; border-radius: 4px; position: relative; }
.bar { height: 24px; border-radius: 4px; }
.bar-label { line-height: 24px; margin-left: 8px; font-size: 12px; }
```
宽度 = (当前值 / 最大值) × 100%，颜色按科室/医生循环分配。

## 功能细节描述

- 所有统计支持日期范围筛选，默认本月
- 条形图用 CSS 实现，标注具体数值
- 管理员看全部数据，医生自动过滤为自己的数据
- 数据为空时显示"暂无数据"占位
- 不引入第三方图表库（Chart.js / ECharts 等）
- 首页指标卡片点击可跳转对应详情
- 统计查询使用 MyBatis 编写 SQL，在 Mapper XML 中实现
